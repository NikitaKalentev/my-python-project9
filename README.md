# Анализ событий информационной безопасности

## Описание проекта
Анализ и визуализация событий информационной безопасности из JSON файла с использованием Python, Pandas, Matplotlib и Seaborn.

## Требования
- Python 3.7+
- Библиотеки: pandas, matplotlib, seaborn
